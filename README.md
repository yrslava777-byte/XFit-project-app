# XFit-app-project
Мобильное приложения для контроля питания и подсчета КБЖУ

## Название
XFit

## Цель
Мобильное приложение для контроля питания и подсчета КБЖУ. Помогает пользователю следить за рационом, рассчитывать суточную норму калорий и вести дневник питания.

## Функции MVP
- Настройка профиля и расчёт индивидуальной нормы КБЖУ
- Поиск продуктов в справочнике
- Добавление записей в дневник питания (завтрак, обед, ужин, перекус)
- Отображение дневного прогресса и остатка калорий
- Просмотр истории записей и редактирование

## Стек технологий

- Kotlin

- Room (SQLite) 
- Clean Architecture + MVVM
